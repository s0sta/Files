# ⚡ البدء السريع - ERPNext v15

## الطريقة السريعة جداً (3 أسطر فقط):

```bash
wget https://your-server.com/install_erpnext_v15.sh
chmod +x install_erpnext_v15.sh
sudo bash install_erpnext_v15.sh
```

---

## 📝 ثم أجب على الأسئلة:

```
1. Enter Frappe username: اكتب اسم المستخدم (أو اترك فارغ للافتراضي)
2. Enter MySQL root password: ادخل كلمة مرور قوية
3. Enter timezone: اترك Asia/Riyadh
4. Enter site name: اترك site1.local
```

## 🔧 الأوامر المهمة بعدها:

